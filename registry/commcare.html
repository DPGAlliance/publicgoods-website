CommCare

CommCare is a digital health platform that allows users to rapidly configure and deploy offline first, custom data collection and service delivery support applications which feature case management for frontline workers without any tech or coding expertise.

Website: https://www.dimagi.com/commcare

Type of Digital Public Good

  • Open content
  • Open data
  • ✅  Open software: Source Code Repository
  • Open standard
  • Open AI model
